# Medidor de Glicose

Aplicativo desenvolvido em Apache Cordova para armazenar localmente, em dispositivos Android, os valores das medições dos índices de glicose

## Tecnologias Utilizadas
- Apache Cordova
- HTML
- CSS
- Javascript

## Tecnologias de outros autores
